Project Overview

This project implements a Bull Call Spread options trading strategy analysis for Reliance Industries using real-time market data from NSE India. The analysis includes option pricing using Black-Scholes model, Monte Carlo simulation, and comprehensive profit/loss analysis.

Key Components

  • Real-time market data analysis from NSE India
  • Black-Scholes option pricing model implementation
  • Monte Carlo simulation for option pricing
  • Put-Call parity validation
  • Profit/Loss analysis and break-even calculations
  • Visual payoff diagram generation

Bull Call Spread Strategy

Strategy Components

  • Buy one call option at ATM strike price
  • Sell one call option at OTM strike price
  • Limited risk and reward profile
  • Suitable for moderately bullish markets

Mathematical Models

Black-Scholes Model

The Black-Scholes model is a mathematical model for pricing European-style options. The formula for a call option is:

C = S·N(d₁) - K·e^(-rT)·N(d₂)

where:

  • d₁ = [ln(S/K) + (r + σ²/2)T] / (σ√T)
  • d₂ = d₁ - σ√T
  • S = Current stock price
  • K = Strike price
  • r = Risk-free interest rate
  • T = Time to maturity
  • σ = Volatility of the stock
  • N(·) = Cumulative distribution function of the standard normal distribution

Monte Carlo Simulation

We use Monte Carlo simulation to estimate option prices by simulating multiple possible paths of the underlying asset price:

S_T = S₀ * exp((r - σ²/2)T + σ√T * Z)

where Z is a standard normal random variable.
