Abstract

This report analyzes a Bull Call Spread strategy for Reliance Industries (RELIANCE.NS) using real option chain data from NSE India (April 25, 2025, expiry May 29, 2025). We derive Black-Scholes and Monte Carlo pricing models from first principles, validate Put-Call Parity, and assess profitability with a payoff diagram and profit table. The strategy, buying a call at 1300 and selling at 1310, yields a break-even of 1305.60, maximum profit of 4.40, and maximum loss of 5.60. The analysis combines rigorous financial mathematics with practical insights, confirming the strategy's suitability for moderately bullish markets.

Option Pricing Models

Black-Scholes Model Implementation


def black_scholes_call_put(S, K, T, r, sigma):
    """
    Calculate Black-Scholes call and put option prices
    
    Parameters:
    S: Current stock price
    K: Strike price
    T: Time to maturity (in years)
    r: Risk-free interest rate
    sigma: Volatility
    
    Returns:
    call_price, put_price: Option prices
    """
    d1 = (np.log(S/K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    
    call_price = S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)
    put_price = K * np.exp(-r * T) * norm.cdf(-d2) - S * norm.cdf(-d1)
    
    return call_price, put_price

Monte Carlo Simulation Implementation


def monte_carlo_option_price(S, K, T, r, sigma, n_simulations=10000):
    """
    Calculate option price using Monte Carlo simulation
    
    Parameters:
    S: Current stock price
    K: Strike price
    T: Time to maturity (in years)
    r: Risk-free interest rate
    sigma: Volatility
    n_simulations: Number of simulation paths
    
    Returns:
    call_price: Estimated call option price
    """
    np.random.seed(42)  # For reproducibility
    Z = np.random.normal(0, 1, n_simulations)
    
    # Simulate stock price at maturity
    S_T = S * np.exp((r - 0.5 * sigma**2) * T + sigma * np.sqrt(T) * Z)
    
    # Calculate payoffs
    payoffs = np.maximum(S_T - K, 0)
    
    # Discount and average payoffs
    call_price = np.exp(-r * T) * np.mean(payoffs)
    
    return call_price

Mathematical Models

Black-Scholes Model

The Black-Scholes model is a mathematical model for pricing European-style options. The formula for a call option is:

C = S·N(d₁) - K·e^(-rT)·N(d₂)

where:

  • d₁ = [ln(S/K) + (r + σ²/2)T] / (σ√T)
  • d₂ = d₁ - σ√T
  • S = Current stock price
  • K = Strike price
  • r = Risk-free interest rate
  • T = Time to maturity
  • σ = Volatility of the stock
  • N(·) = Cumulative distribution function of the standard normal distribution

Monte Carlo Simulation

We use Monte Carlo simulation to estimate option prices by simulating multiple possible paths of the underlying asset price:

S_T = S₀ * exp((r - σ²/2)T + σ√T * Z)

where Z is a standard normal random variable.

Put-Call Parity

The put-call parity relationship must hold for European options:

C + K·e^(-rT) = P + S

where:

  • C = Call option price
  • P = Put option price
  • K = Strike price
  • S = Current stock price
  • r = Risk-free rate
  • T = Time to maturity
